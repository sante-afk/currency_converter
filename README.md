
## This application allows you to convert an amount from one currency to another using current exchange rates.

#### Features

>Currency conversion via the command line

>HTTP API for getting exchange rates and performing conversions

>Support for popular currencies (USD, EUR, GBP, etc.)

>Easy installation and launch

#### Launch
```bash
git clone https://github.com/sante-afk/currency_converter.git
```
Install dependencies
```js
npm install
```
